About This Application

This application provides an automated pipeline for fine-tuning and retraining of object detection models, focusing on YOLO and RF-DETR architectures.

Supported Features:
  • Upload and manage custom datasets
  • Configure model training parameters
  • Fine-tune YOLO and RF-DETR models
  • Track experiments with MLFlow
  • Orchestrate workflows with Dagster
  • Visualize training metrics and results
Supported Models:
YOLOv5s YOLOv5m YOLOv5l YOLOv5x YOLOv8n YOLOv8s YOLOv8m YOLOv8l YOLOv8x RF-DETR-R50 RF-DETR-R101
